# EF-CUSTOM - Custom EF Management

## Overview
Custom emission factor management system for organization-specific or supplier-provided factors.

## Category
Emission Factor Management

## Description
Custom EF management allows organizations to upload and manage their own emission factors, including supplier-specific data and process-specific measurements. The system validates custom factors against industry standards and provides version control and audit trails for all custom data.

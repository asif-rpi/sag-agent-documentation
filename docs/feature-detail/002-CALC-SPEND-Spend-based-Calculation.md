# CALC-SPEND - Spend-based Calculation

## Overview
Spend-based calculation method for estimating carbon emissions using financial expenditure data as a proxy when direct activity data is unavailable.

## Category
Core Calculation

## Description
Spend-based calculation provides a quick estimation approach for carbon emissions by using purchase values and economy-wide emission factors. This method is particularly useful for initial assessments or when detailed activity data is not yet available. The calculation multiplies purchase value by an Environmentally Extended Input-Output (EEIO) emission factor to derive CO2e estimates. While less accurate than supplier-specific or process-specific methods, it serves as a valuable starting point for companies beginning their carbon accounting journey.

# REPORT-CHARTS: Emission Trends

## API Basic Information

| Item | Content |
|------|---------|
| Unique ID | REP-TREND |
| API Name | Emission Trends |
| Type | GET |
| Endpoint | /api/reports/trends |
| Category | Reporting, Dashboard & Audit |
| Status | Planned |

---

## Summary

Get monthly/yearly trend data for charts. Interactive charts and trend visualization for carbon emission data.

---

## Chart Types

| Chart | Use Case |
|-------|----------|
| Line Chart | Trend over time |
| Bar Chart | Compare categories |
| Pie Chart | Distribution of emissions |
| Heat Map | Geographic distribution |
| Stacked Bar | Scope breakdown over time |
| Area Chart | Cumulative emissions |

---

## Dashboard Widgets

| Widget | Description |

| Top Categories | Bar chart |

## Interactive Features

| Feature | Description |
|---------|-------------|
| Drill-down | Click to see detail |

| Export | Save chart as image |

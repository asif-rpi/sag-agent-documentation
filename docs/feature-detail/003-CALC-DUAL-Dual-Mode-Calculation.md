# CALC-DUAL - Dual Mode Calculation

## Overview
Dual mode calculation system that supports both mass-based and spend-based methods for comprehensive carbon emission assessment.

## Category
Core Calculation

## Description
Dual mode calculation enables users to switch between mass-based and spend-based calculation methods depending on data availability and accuracy requirements. This flexible approach allows for hybrid calculations where some data points use direct measurements (mass-based) while others use financial proxies (spend-based). The system automatically selects the most appropriate method based on data quality and provides transparent reporting of which method was used for each calculation.

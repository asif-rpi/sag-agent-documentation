# EF-DEFAULT - Default EF Database

## Overview
Default emission factor database providing standardized factors for carbon calculations.

## Category
Emission Factor Management

## Description
Default EF database contains comprehensive emission factors from authoritative sources including IPCC, DEFRA, and Japan MOE. The system automatically selects appropriate factors based on location, industry, and activity type, ensuring consistent and reliable carbon calculations.

# INPUT-MANUAL - Manual Form Entry

## Overview
Manual data entry forms for inputting carbon emission data when automated sources are unavailable.

## Category
Smart Input System

## Description
Manual form entry provides user-friendly web forms for entering individual data points or small datasets. Forms include validation, auto-save functionality, and guided input fields to ensure data accuracy. This feature supports companies in the early stages of carbon accounting or for entering data from sources that cannot be automated.

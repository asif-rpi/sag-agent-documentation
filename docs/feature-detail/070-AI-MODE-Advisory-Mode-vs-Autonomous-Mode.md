# AI-MODE: Advisory Mode vs Autonomous Mode

## Feature Basic Information

| Item | Content |
|------|---------|
| Unique ID | AI-MODE |
| Feature Name | Advisory Mode vs Autonomous Mode |
| Category | AI Agentic Intelligence |
| Priority | High |
| Implementation Phase | Phase 3-4 |

---

## Purpose

Two operating modes — Advisory (AI suggests, human approves) vs Autonomous (AI auto-executes within pre-approved parameters).

---

## Mode Comparison

| Mode | AI Role | Human Role | Phase |
|------|---------|-----------|-------|
| MVP | Calculate, report | Reviews all | Phase 1 |
| Phase 2 | Suggest alternatives | Approves recommendations | Phase 2 |
| Advisory | Predict disruptions | Sets parameters | Phase 3 |
| Autonomous | Auto-reroute, auto-switch | Monitors exceptions | Phase 4 |

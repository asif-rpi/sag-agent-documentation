# REPORT-TOTAL - Total CO₂ Emission Report

## Overview
Comprehensive total carbon emission reporting across all scopes and categories.

## Category
Reporting & Dashboard

## Description
Total CO₂ emission report provides aggregated carbon footprint data including Scope 1, 2, and 3 emissions. The report includes year-over-year comparisons, trend analysis, and breakdowns by business unit, product line, and supplier.

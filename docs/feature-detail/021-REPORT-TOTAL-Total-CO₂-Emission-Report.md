## Purpose

Comprehensive total carbon emission reporting across all scopes and categories. Provides aggregated carbon footprint data for decision-making.

---

## Report Contents

| Section | Description |
|---------|-------------|
| Executive Summary | High-level totals and trends |
| Scope 1 | Direct emissions |
| Scope 2 | Energy-related indirect emissions |
| Scope 3 | Other indirect emissions |
| YoY Comparison | Year-over-year changes |
| Breakdown | By business unit, product, supplier |

# INPUT-CSV - CSV Upload

## Overview
CSV file upload functionality for bulk data import and processing of carbon emission data.

## Category
Smart Input System

## Description
CSV upload allows users to import large datasets containing order, shipment, and activity data in comma-separated value format. The system supports customizable column mapping, data validation during upload, and automatic processing of multiple records. This feature is essential for companies with existing data in spreadsheet format and serves as a bridge for those not yet ready for full API integration.

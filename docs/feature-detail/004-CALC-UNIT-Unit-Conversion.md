# CALC-UNIT - Unit Conversion

## Overview
Automatic unit conversion system for consistent carbon emission calculations across different measurement units.

## Category
Core Calculation

## Description
Unit conversion ensures all input data is standardized to appropriate units before carbon calculations. The system handles conversions between various units commonly used in supply chain and logistics data, including weight (kg, tonnes, lbs), volume (liters, gallons), distance (km, miles), and energy (kWh, MJ). This prevents calculation errors and ensures accurate emission factors are applied consistently across all data sources.

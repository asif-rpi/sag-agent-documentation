| Item | Content |

| Category | Route & Supply Chain Optimization |
| Priority | High |
| Implementation Phase | Phase 3 |

---

## Purpose

Combine multiple small shipments into full container loads to reduce emissions.

---

## Consolidation Benefits

| Benefit | Impact |
|---------|--------|
| Reduced Trips | 30-50% fewer shipments |
| Lower Carbon | 20-40% reduction |
| Cost Savings | 15-25% lower cost |
| Efficiency | Better capacity use |

---

## Consolidation Rules

| Rule | Description |
|------|-------------|
| Destination | Similar delivery area |
| Timing | Compatible transit times |
| Product | Compatible goods |
| Temperature | Same requirements |

---

## Algorithm

| Step | Description |
|------|-------------|
| 1 | Identify consolidation opportunities |
| 2 | Calculate combined emissions |
| 3 | Compare vs separate shipments |
| 4 | Generate recommendation |
| 5 | Track consolidation rate |
